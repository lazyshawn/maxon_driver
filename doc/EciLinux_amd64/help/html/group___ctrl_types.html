enum e_SETTINGS_FLAGS

Controller settings flags Bit coded information, resulting value can be any combination of values below.

Enumerator:
ECI_SETTINGS_FLAG_NONE 

No flags used.

ECI_SETTINGS_FLAG_POLLING_MODE 

If set the hardware is initialized in polling mode, that means no interrupt is assigned and the interrupt line is not used.

To use the recommended interrupt driven mode set this flag resp. bit to zero. Please also refer to Interrupt Mode vs. Polling Mode.

ECI_SETTINGS_FLAG_NO_FW_VERIFY 

If set the the board is opened without verifying the ECI firmware after downloading it.

This switch can be used if start time of the board shall by reduced.

ECI_SETTINGS_FLAG_NO_FW_DOWNLOAD 

If set the the board is opened without downloading the ECI firmware.

This flag is only used by maintenance programs and must not be set by any ECI application

enum e_STOP_FLAGS

Controller stop flags.

Enumerator:
ECI_STOP_FLAG_NONE 

No flags used.

ECI_STOP_FLAG_RESET_CTRL 

If set an ongoing transmission is aborted and the controller configuration is reset.

(RESET)
If not set the controller is stopped after an ongoing transmission, the configuration is kept (STOP).

ECI_STOP_FLAG_CLEAR_TX_FIFO 

If set all entries from TX FIFO are cleared
If not set all entries are kept in TX FIFO.

ECI_STOP_FLAG_CLEAR_RX_FIFO 

If set all entries from RX FIFO are cleared.


If not set all entries are kept in RX FIFO.

Note:
Please note if RX FIFO clearing is not enabled while re-starting the controller. The timestamps are also not reset upon re-start of the controller.
