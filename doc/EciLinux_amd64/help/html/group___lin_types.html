enum e_LINMSGFLAGS

LIN message information flags (used by <LINMSGINFO.Bytes.bFlags>).

Bit coded information, resulting value can be any combination of values below.

Enumerator:
ECI_LIN_MSGFLAGS_DLC 

Data length code.

ECI_LIN_MSGFLAGS_OVR 

Data overrun flag (RX direction only).

ECI_LIN_MSGFLAGS_SOR 

Sender of response.

Set to send slave buffer data upon reception if LIN ID in slave mode. To disable a slave buffer send a slave message without setting this flag.

ECI_LIN_MSGFLAGS_ECS 

Enhanced checksum (according to LIN Spec. 2.0).

ECI_LIN_MSGFLAGS_IDO 

ID only.

Set to send a LIN frame without data in master mode. Upon reception this flag indicates that no slave responded to this LIN ID.

ECI_LIN_OPMODE_CACHED_STATUS 

The status will be cached within the driver which leads to very fast ECIDRV_CtrlGetStatus calls.

However you may need to set a status update rate with ECIDRV_CtrlSetStatusUpdateRate.
