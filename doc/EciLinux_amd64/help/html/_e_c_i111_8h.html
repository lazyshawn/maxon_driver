Detailed Description

ECI API function redefintions for IXXAT Interfaces: CAN-IB2x0/PCIe (104), CAN-IB4x0/PCI (104), CAN-IB6x0/PCIe FD (104) and CAN-IB8x0/PCI FD (104).

Headerfile of the ECI ( embedded / realtime communication interface), a generic library for IXXAT hardware interfaces.
