Detailed Description

OS ECI functions.

Mapping of the underlying operation system functions to simplify porting the ECI driver to another operating system.

Acquires a mutual lock.

Parameters:

Return values:
DWORD If the function succeeds, the return value is TRUE. If the function fails, the return value is FALSE. To get extended error information, call OSECI_GetLastError.

DWORD OSECI_WaitForSingleObject ( ECI_HANDLE  hEvent,
DWORD  dwTimeout 
)

Wait until either the specified object is in the signaled state, or until the time-out interval elapses.

Parameters:
hEvent Handle to wait for get signaled
dwTimeout Specifies the time-out interval, in milliseconds [ms]. If the interval elapses, the function returns, even if the object's state is non signaled.
Return values:
DWORD OSECI_WAIT_OBJECT_0 Success. The specified object�s state is signaled.
OSECI_WAIT_TIMEOUT Failure. The time-out interval elapsed, and the object's state is non signaled.
OSECI_WAIT_FAILED Failure. Waiting failed maybe due to an invalid handle. To get extended error information, call OSECI_GetLastError.
